# softroller

### 概要
柔らかいマテリアルを用いてゲームのコントローラを作る.

### ゲーム
ヘリコプターを操作して飛んでくる弾丸を避ける.

### コントローラ
Arduino, 圧力センサFSR406, 3軸加速度センサKXR94-2050, StandardFirmata.  
圧力センサは決定ボタン, 3軸加速度センサはヘリコプターの操作に使用.
